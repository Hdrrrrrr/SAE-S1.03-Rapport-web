Installation & Déploiement

Processus de création de la machine virtuelle et installation d'Ubuntu

1. Création de la VM

Dans VirtualBox, nous initialisons une nouvelle machine virtuelle :

  • Nom : Ubuntu (pour identifier la distribution)
  • Image ISO : Sélection de ubuntu-24.04.3-desktop-amd64.iso
  • Type : Linux / Ubuntu (64-bit)
Note : L'option "Skip unattended install" est cochée pour maîtriser manuellement chaque étape du partitionnement.

4. Tutoriel Vidéo : Processus d'Installation

Cette démonstration montre la création de la VM, le partitionnement et le premier démarrage d'Ubuntu Desktop.

2. Allocation des Ressources

Afin de supporter l'environnement GNOME et nos outils de développement, nous avons alloué des ressources conséquentes :

Mémoire RAM : 8192 MB (8 Go)

Indispensable pour la fluidité de l'interface graphique et l'exécution simultanée du serveur Apache.

Processeurs (CPU) : 4 cœurs

Permet une gestion efficace du multitâche sans ralentir la machine hôte.

Disque Dur : 30 Go

Stockage en allocation dynamique pour optimiser l'espace sur l'hôte.

3. Finalisation du Déploiement

Configuration NAT

Le réseau est configuré en mode NAT avec redirection de port (8080 -> 80) pour l'accès web externe.

Système Prêt

Après l'installation, le système est prêt pour la phase d'administration des comptes et services.
